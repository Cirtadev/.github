# Cirtadev

Cirtadev GmbH is a Switzerland-based software company building reliable, privacy-aware web platforms and IT solutions for modern businesses.

We design, build, and operate products that focus on reliability, observability, real-world usability, and maintainable software.

## Products

- **Cirtaflow**: Camera-augmentation and analytics platform for turning video streams into actionable insights.
- **Cryptosan**: Crypto market monitoring platform that filters market noise and surfaces relevant signals.
- **Otalan**: OTA update tooling and infrastructure for mobile applications.

## Open Source

We maintain developer tools and libraries used across our products, including `@otalan/cli` and `@otalan/sdk`.
